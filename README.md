# dataset_toyota


Para iniciar proyecto

Primero crear un entorno conda

```bash
conda create -n "env_name" python=3.11
```

```bash
conda activate "env_name"
```

```bash
pip install -e ".[dev]"
```

```bash
pip install poetry
```

```bash
poetry install
```

Then, start the Dagster UI web server:

```bash
dagster dev
```

En el buscador ingresar a http://localhost:3000
